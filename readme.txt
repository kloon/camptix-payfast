PayFast Payment Gateway for CampTix plugin

== Description ==

South African PayFast Payment Gateway for the CampTix plugin.

Take payments in ZAR through PayFast using the CampTix plugin. CampTix plugin needs to be installed and activated for the PayFast payment gateway to work.

== Installation ==

1. Upload `camptix-payfast` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to `Tickets -> Setup` in your WordPress admin area to set the currency to ZAR and activate the PayFast gateway.
